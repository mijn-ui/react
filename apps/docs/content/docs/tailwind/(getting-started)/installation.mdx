<Steps>
<Step>
### Install Tailwind.CSS

This is just a basic setup. For complete installation instructions, please refer to the [Tailwind CSS installation docs](https://tailwindcss.com/docs/installation).

```bash title="Terminal"
  npm install -D tailwindcss
  npx tailwindcss init 
```

</Step>

<Step>
### Add Css Variables

These are the default themes provided by mijn-ui. If you would like to create your own theme, learn more about themes [here](/docs/tailwind/theming).

```json doc-gen:file
{
  "file": "./examples/html/index.css",
  "codeblock": {
    "lang": "css",
    "meta": "title=\"index.css\""
  }
}
```

</Step>

<Step>
### Add TailwindCss config
 ```json doc-gen:file
{
  "file": "./examples/html/tailwind.config.js",
  "codeblock": {
    "lang": "js",
    "meta": "title=\"tailwind-config.js\""
  }
}
```
</Step>

<Step>
  ### That's it
  
  You can now start adding components to your project. Check out the [components](/docs/tailwind/components/accordion) section to learn more about the available components.

</Step>
</Steps>
