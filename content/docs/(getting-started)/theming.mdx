## List of variables

---

#### Main Colors

Primary colors for backgrounds, text, and borders across the UI.

```css
{
  --main: 210 14% 95%; /* slate-100 */
  --main-text: 0 0% 0%; /* black */
  --main-border: 0 0% 83%; /* neutral-300 */
}
```

#### Surface Colors

For panels, cards, and other contained elements.

```css
  {
  --surface: 0 0% 98%; /* neutral-50 */
  --surface-text: 0 0% 0%; /* black */
}
```

#### Accent colors

For interactive elements like hover effects.

```css
{
  --accent: 0 0% 90%; /* neutral-200 */
  --accent-text: 0 0% 0%; /* black */
}
```

#### Neutral Colors

Muted tones for secondary content, providing balanced visual hierarchy.

```css
{
  --neutral: 0 0% 83%; /* neutral-300 */
  --neutral-text: 0 0% 25%; /* neutral-700 */
}
```

#### Border and Ring Colors

```css
{
  --border: 0 0% 83%; /* neutral-300 */
  --ring: 0 0% 83%; /* neutral-300 */
}

```

#### Input Border Color

Used in form fields like Input, Select, and Textarea.

```css
{
  --input-border: 0 0% 0%; /* black */
}

```

---

### Status Colors

status colors for components such as `Alert`, `Input`, `Button`, `Progress`, `Indicator`, etc...

- `background`: Background color of the component.
- `text`: Text color when the component background is outline.
- `filled-text`: Text color when the component background is filled.

#### info

```css
{
  --info: 216 76% 52%; /* blue-500 */
  --info-text: 219 64% 43%; /* blue-800 */
  --info-filled-text: 0 0% 100%; /* white */
}
```

#### warning

```css

  --warning-filled-text: 0 0% 0%; /* white */
}
```

#### danger

```css

  --danger-filled-text: 0 0% 100%; /* white */
```


## Adding New Colors

To add new colors, you need to add them to your CSS file and to your tailwind.config.js file.

<Tabs items={['global.css', 'tailwind.config.js']}>
  <Tab value="global.css">
  ```css
  :root {
    --card: 0 0% 98%; /* neutral-50 */
    --card-text: 0 0% 0%; /* black */
  }

.dark {
--card: 0 0% 7%; /_ neutral-925 _/
--card-text: 0 0% 96%; /_ neutral-100 _/
}

````
  </Tab>

  <Tab value="tailwind.config.js">
  ```js
  module.exports = {
    theme: {
      extend: {
        colors: {
          card: "hsl(var(--card))",
          "card-foreground": "hsl(var(--card-foreground))",
        },
      },
    },
  }
````

  </Tab>
</Tabs>

You can now use the new colors in your components.

```tsx
<div className="bg-card text-card-foreground">Hello</div>
```

### Other Color Formats

You can also use other color formats if you prefer. Tailwind CSS supports various color formats including RGB, RGBA, and HSL.

For more information on using different color formats, refer to the [Tailwind CSS documentation](https://tailwindcss.com/docs/customizing-colors#using-css-variables).
